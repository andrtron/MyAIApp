# Ayla AI Assistant

Мобильное Android-приложение: оффлайн ассистент с голосовой активацией, поиском в интернете по запросу и TTS-ответами.

## Особенности
- Активация по фразе «Айла»
- Остановка «Айла, стоп»
- Голосовой ввод (Vosk)
- Локальная логика + онлайн-поиск
- TTS (женский голос)
- Поддержка Bluetooth-наушников
- Минималистичный интерфейс: история чата

## Сборка
Сборка происходит автоматически через GitHub Actions при пуше в `main`.  
APK будет доступен в разделе **Actions → Build Android APK → Artifacts**.

## Подпись (опционально)
Для подписанного релиза нужно настроить секреты:
- `KEYSTORE_BASE64` — base64 версии вашей `.jks` (кодируется локально)
- `KEYSTORE_PASSWORD`
- `KEY_ALIAS`
- `KEY_PASSWORD`

## Как получить APK
1. Сделать пуш или вручную запустить workflow.  
2. Перейти в Actions, скачать артефакт (APK).  
3. Установить на Android-устройство.
